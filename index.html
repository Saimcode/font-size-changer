Font Size Changer

This is a simple Font Size changer program that will let you change the font in 3 different sizes. There are 3 buttons on the top right corner of the webpage. You can click them to change the font size.
1. The first button will transform the font size to 16px.
2. The second button will transform the font size to 24px.
3. The thrid button will transform the text to 32px.

This program is made with HTML, CSS & JavaScript.
